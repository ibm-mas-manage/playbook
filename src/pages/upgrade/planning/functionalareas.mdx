### These are the functional areas detail.


<table>
  <tr>
    <th>Functional Area</th>
    <th>Detail</th>
  </tr>

  <tr>
    <td>Attached Docs</td>
    <td>
     <pre>Fully Supported.</pre>
          <pre>File System</pre>
	   <pre>   If using the file system mount, the same file system mount can be configured in MAS through volume mount in OpenShift. </pre>
           <pre>   No change is needed for the Manage declines configuration, if the mount point is same as before.</pre>
          <pre>Object Storage(S3)</pre>
	   <pre>   No change needed to bring attached documents.</pre>
           <pre>   Object storage server public certificate may need to be imported.</pre>
    </td>
   </tr>

   <tr>
    <td>Reporting</td>
    <td>
     <pre>BIRT supported.</pre>
     <pre>Cognos will not be supported in the initial version(8.0).</pre>
	   
    </td>
   </tr>
   
   <tr>
    <td>Industry Solutions/Add-ons</td>
    <td>
     <pre>All Industry Solutions/Add-ons will be supported.</pre>
    </td>
   </tr>

   <tr>
    <td>Anywhere</td>
    <td>
     <pre>Updated Mobile Apps will be available for download.</pre>
    </td>
   </tr>

   <tr>
    <td>Maximo Mobile: Work Centers in disconnected mode</td>
    <td>
     <pre>Initial version will contain one application, three functions (Tecchnician, Inspections and Assist).</pre>
    </td>
   </tr>

   <tr>
    <td>Scheduler, Calibration, Linear</td>
    <td>
     <pre>Fully supported.</pre>
    </td>
   </tr>

  <tr>
    <td>Migration Manager</td>
    <td>
     <pre>Fully supported.</pre>
     <pre>All pending migration packages should be migrated prior to upgrade (prerequisite).</pre>
     <pre>Configuration options will be provided during deployment for persistent volume mount for file-based packages. </pre>
     <pre>Automation APIs will be available. Supports the automation pf package creation and deployment. </pre>
    </td>
   </tr>

  <tr>

     <pre>All languages(25) will be supported.</pre>
    </td>
   </tr>

  <tr>
    <td>SAP/Oracle Integration</td>
    <td>
     <pre>Fully Supported.</pre>
    </td>
   </tr>

   <tr>
